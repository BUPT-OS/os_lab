1. 在微信群内置顶的共享文档中，填写个人信息，用于创建每个人的账号。初始账号密码为所填写手机号；

## 考核要求和结果
1. 根据以往参与考核人员的情况，争取在2-3周内完成所有的lab，并在测试平台上通过所有的测试；

3. 独立自主完成，不得抄袭。
